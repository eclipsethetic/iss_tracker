# iss_tracker

В этом проекте я сделал небольшую программу отслеживающую траекторию движения МКС на орбите вокруг Земли.
Программа получает данные в реальном времени. Значок МКС передвигается по карте мира используя реальные координаты.

#ЦЕЛИ:

Научиться получать реальные данные от сторонних сервисов через API.
Преобразовывать их в нужный формат.
Научиться основам библиотеки turtle.


#ЧТО ИСПОЛЬЗОВАЛ:

Python 3.14.3
requests
turtle


#ЧЕМУ Я НАУЧИЛСЯ:
